# xiyi
上门洗衣_洗衣洗鞋_洗衣洗鞋系统_上门服务_洗鞋小程序_洗鞋洗衣_洗鞋项目_洗衣店智能管理系统_e洁洗衣_洗衣预约_洗衣平台

# 《洗鞋系统使用说明》

支持会员、优惠券、招商加盟、合伙人分销、分类管理、购物车、视频展示、站点门店下单配送费，站点管理等功能，新人优惠券，对接打印机

# 技术架构

java后台技术springboot+jpa+mysql
用户端uniapp
管理端vue +elementUI
